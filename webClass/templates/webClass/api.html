api documentation

/add-project

add a project folder and database entry

headers

X-CSRFToken : < csrf token >

body

name : < new project name >
desc : < new project description >

/set-desc

changes the description of a project

headers

X-CSRFToken : < csrf token >

body

name : < project name >
desc : < new project description >

/set-id

changes the id of a project

headers

X-CSRFToken : < csrf token >

body

name : < project name >
id : < new project id >

/delete-project

sends a project to the 'archived' file

headers

X-CSRFToken : < csrf token >

body

name : < project name >

/clear-archive

deletes a file from the 'archived' folder
note: setting 'id' to 'all' will clear the whole archive

headers

X-CSRFToken : < csrf token >

body

id : < project id >

notes:
all endpoints are formatted as < base url >/webClass/api/< endpoint >
all endpoints require an x-csrf token which can be provided upon request
